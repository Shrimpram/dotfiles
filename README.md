# My Dotfiles
These are my dotfiles.

## Installation
### Setup
1. `cd ~`
2. git clone this repo
3. `cd dotfiles`
4. `./install`

### Nix
1. Install [nix](https://nixos.org)
2. Install [home manager](https://github.com/nix-community/home-manager)
3. Run `home-manager switch`

### Nvim
1. Run `./bootvim.sh`
